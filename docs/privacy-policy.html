隐私政策

适用于“Edge 会话恢复工具”
最后更新日期：2026-04-09

本扩展仅用于在本地设备上保存和恢复 Microsoft Edge 会话状态，不提供账号系统，不要求登录，也不会把浏览数据上传到远端服务器。

1. 我们处理哪些数据

为了实现会话恢复功能，本扩展会在本地处理以下信息：

  • 标签页标题、URL、活动状态、固定状态和标签组信息。
  • 窗口的位置、大小、焦点状态和窗口类型。
  • 保存在浏览器本地存储中的快照、恢复事务、诊断状态和设置。

2. 这些数据如何被使用

本扩展使用 tabs、windows、tabGroups 和 storage 权限，仅用于：

  • 保存当前浏览会话的本地快照。
  • 在关闭最后一个普通窗口后保留最近一次可恢复状态。
  • 恢复窗口、标签页和标签组布局。
  • 在扩展弹窗中显示状态、计数和恢复结果。

3. 是否会上传或共享数据

不会。本扩展不包含远端后端服务，不会把浏览记录、快照内容、诊断信息或任何恢复数据上传到第三方服务器，也不会与广告网络、分析平台或其他第三方共享这些数据。

4. 账号、分析与追踪

本扩展不提供账号系统，不要求登录，不使用广告标识符，不集成第三方分析 SDK，也不进行跨站追踪。

5. 数据保留与删除

相关数据仅保存在用户设备本地，直到以下情况之一发生：

  • 用户完成恢复流程。
  • 用户选择忽略恢复。
  • 用户卸载扩展或清除浏览器中的扩展数据。

6. 政策更新

如果未来新增远端服务、账号系统或分析能力，本页面会同步更新。

7. 联系方式

如需进一步说明，请通过本扩展的 GitHub 仓库或发布页面联系维护者。
