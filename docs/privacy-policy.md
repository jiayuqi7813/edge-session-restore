# 隐私政策

最后更新日期：2026-04-09

本隐私政策适用于“Edge 会话恢复工具”（以下简称“本扩展”）。

## 1. 我们收集什么

本扩展不创建账号，不要求登录，也不会主动收集用户姓名、邮箱、电话号码或其他身份信息。

本扩展仅在用户设备本地处理与会话恢复相关的数据，可能涉及：

- 标签页标题、URL、活动状态、固定状态和分组信息。
- 窗口的位置、大小、焦点状态和类型。
- 本地存储中的快照、恢复事务、诊断状态和设置。

这些数据仅用于实现会话保存与恢复功能。

## 2. 我们如何使用这些数据

本扩展读取 `tabs`、`windows` 和 `tabGroups` 相关信息，用于：

- 保存当前浏览会话的本地快照。
- 在关闭最后一个普通窗口后保留最近一次可恢复状态。
- 恢复窗口、标签页和标签组布局。
- 显示弹窗中的状态、计数和恢复结果。

本扩展使用 `storage` 权限仅在本地保存这些状态，不用于远端同步。

## 3. 我们是否把数据发送到服务器

不会。

本扩展不包含远端后端服务，不会把浏览记录、快照内容、诊断信息或任何恢复数据上传到第三方服务器。

## 4. 第三方共享

不会。

本扩展不出售、不出租，也不向广告网络、分析平台或其他第三方共享用户浏览数据。

## 5. 账号与追踪

本扩展：

- 不提供账号系统。
- 不要求登录。
- 不使用广告标识符。
- 不集成第三方分析 SDK。
- 不进行跨站追踪。

## 6. 本地存储

本扩展会在用户设备本地使用浏览器提供的存储能力保存必要状态，例如：

- 最近一次稳定快照。
- 待恢复状态。
- 恢复事务。
- 诊断信息和设置。

用户可以通过卸载扩展或清除浏览器扩展数据来删除这些本地数据。

## 7. 权限说明

本扩展申请的权限仅用于恢复功能：

- `storage`：保存本地快照和设置。
- `tabs`：读取和恢复标签页。
- `windows`：读取和恢复窗口。
- `tabGroups`：读取和恢复标签组。

## 8. 数据保留

本扩展会在本地保留最近一次可恢复快照和相关状态，直到：

- 用户完成恢复流程。
- 用户选择忽略恢复。
- 用户卸载扩展或清除本地扩展数据。

## 9. 儿童隐私

本扩展不面向儿童，也不以收集儿童个人信息为目的。

## 10. 政策变更

如果本扩展未来新增远端服务、账号系统或分析功能，本隐私政策会相应更新。

## 11. 联系方式

如需关于本扩展隐私政策的说明，可通过 GitHub 仓库或发布页面联系维护者。
