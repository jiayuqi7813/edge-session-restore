# Microsoft Edge Add-ons 上架文案

以下内容可直接复制到 Microsoft Edge Add-ons 后台。字段名按常见提交流程整理，中文为主，必要时保留英文字段建议。

## 短描述

在 macOS 上关闭最后一个 Edge 窗口后，自动保存最近一次可恢复会话，并在工具栏中一键恢复。

## 详细描述

Edge 会话恢复工具是一个面向 Microsoft Edge 的浏览器扩展，专注于 macOS 场景下的会话保护与恢复。

它会在浏览器运行期间持续记录窗口、标签页和标签组的稳定状态；当你关闭最后一个普通窗口时，扩展会自动保留最近一次可恢复快照。之后，你只需点击工具栏图标，就能查看当前状态并恢复上次会话。

这个扩展适合以下场景：

- 临时关闭了整个 Edge 窗口，但还想回到刚才的工作环境。
- 关闭窗口后再次打开 Edge，希望快速恢复标签页、窗口布局和标签组。
- 不想依赖系统级会话恢复，而是希望有一个明确、可控的恢复入口。

扩展会尽量恢复窗口、标签页顺序、标签组与窗口几何信息，但不会承诺恢复所有临时状态，例如滚动位置、未提交表单、媒体播放进度或前进后退历史。

## 功能亮点

- 自动保存最近一次稳定会话快照。
- 关闭最后一个普通窗口后，保留可恢复状态。
- 点击工具栏图标即可查看恢复入口与当前状态。
- 支持窗口、标签页和标签组的重建。
- 对受保护页面自动跳过，避免恢复 edge://、扩展页和 DevTools 等不可恢复内容。
- 对 `file://` 页面提供可控恢复，取决于用户是否允许本地文件访问。
- 具备保护性确认逻辑，避免把旧会话直接塞进已有工作区。

## 权限说明

建议在商店后台使用如下说明，帮助审核人员理解权限用途：

- `storage`：用于在本地保存快照、恢复事务、诊断状态和用户设置。
- `tabs`：用于读取标签页信息并在恢复时重建标签页。
- `windows`：用于读取和恢复窗口布局、窗口状态和焦点。
- `tabGroups`：用于读取和恢复标签组信息。

补充说明：

- 扩展仅在本地设备上工作，不会把浏览数据上传到远端服务器。
- 扩展不包含账号系统，不要求登录。
- 扩展不做广告归因、行为分析或第三方追踪。

## 测试说明

提交审核时可以提供以下测试步骤：

1. 在 macOS 上安装扩展并启用。
2. 打开多个 Edge 窗口和标签页，最好包含一个或多个标签组。
3. 点击最后一个普通窗口左上角的关闭按钮。
4. 重新打开 Edge，并点击工具栏里的扩展图标。
5. 在弹窗中确认会话状态，并点击“恢复上次会话”。
6. 验证窗口、标签页顺序和标签组是否按预期恢复。

如果当前已经有正常工作区，扩展应进入确认模式，而不是直接覆盖当前窗口。

### v1.0.0

- 初始版本。
- 支持关闭最后一个普通窗口后保存会话快照。
- 支持从工具栏一键恢复上次会话。
- 支持窗口、标签页和标签组恢复。

### v1.0.1

- 修复问题：恢复后再次关闭浏览器时，可能持续恢复首次保存的标签页。
- 优化体验：弹窗诊断区新增状态码、快照 ID、捕获原因、最近触发和 Pending 状态。
- 新增能力：支持一键复制调试 JSON，复制受限时自动展开可手动复制的调试内容。
